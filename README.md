# Genesys Cloud CX Automation 🎧 ☁️

Este repositorio contiene scripts y herramientas en **Python** diseñadas para la automatización, gestión y extracción de datos en la plataforma de Contact Center en la nube **Genesys Cloud CX** utilizando sus APIs oficiales.

## 🚀 Objetivo del Proyecto
Optimizar la administración operativa y técnica de Genesys Cloud, eliminando tareas manuales repetitivas y facilitando la auditoría de configuraciones mediante interacciones programáticas.

## 🛠️ Alcance Técnico
* **API Integration:** Consumo de la **Genesys Cloud Platform API** para la automatización de configuraciones y aprovisionamiento masivo.
* **Data & Analytics:** Scripts en **Python** (utilizando el SDK oficial o requests directos) para la extracción y generación de reportes personalizados de métricas, colas o usuarios.
* **Configuración Programática:** Gestión y consulta de recursos del Contact Center (Routing Queues, Users, Roles) a través de código.

## 📋 Requisitos
* Cuenta activa de Genesys Cloud con permisos de integrador/administrador.
* Credenciales de tipo **OAuth Client (Client Credentials)**.
* Python 3.9 o superior.
